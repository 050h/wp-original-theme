焦りは禁物？「待って攻める」が結婚できる女のテクニックだった

皆さんどうもお久しぶりです！@05_0hです
今日は、地上波の録画サーバーを作っていきたいと思います！

概要

使用OS : Linux(Ubuntu v15.10)
チューナー : PX-U1UD v2.0
録画ソフト : chinachu

用意するもの

・てきとうなマシン
・PX-U1UD
・ちょっとした知識

手順

1.Ubuntuをインストール

まずはパソコンにUbuntuをインストールします。
Ubuntuのインストールの仕方はググってください。
PX-U1UDを使用するためのドライバがv3.15以上のカーネルじゃないと対応していないので、
バージョンは15.10が良いと思います。

2.Dockerをインストール

私の友人@wh11e7rueが環境を全部Dockerイメージにしてくれたのでそれを使うのでLinuxにDockerをインストールします。
Dockerのインストールは至って簡単。
sudo apt-get install docker.io
これだとバージョンが古いので、バージョンを上げます
apt-transport-httpsが必要らしいので、インストールしておきます
sudo apt-get install apt-transport-https
鍵の取得とレポジトリを追加します
sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys 36A1D7869245C8950F966E92D8576A8BA88D21E9
sudo sh -c "echo deb https://get.docker.com/ubuntu docker main > /etc/apt/sources.list.d/docker.list"
sudo apt-get update
sudo apt-get install lxc-docker
docker --version
で1.9.1とかになってればおｋ

アンテナをつなげる

PX-U1UDにアンテナ線をさして、パソコンに繋げておいてください。

コンテナを動かす

ソース このコマンドを打つだけ
docker run -d -p 10772:10772 --volume /path/to/recorded:/home/chinachu/recorded --device /dev/dvb/adapter0:/dev/dvb/adapter0 wh11e7rue/chinachu-pxs1ud
これで、エラーがでなければブラウザからlocalhost:10772にアクセスできるはず
※初回は、番組表を取得するのに少し時間がかかります

ポートを開放する

ポートを開放するには、各家庭によって、使っているルーターが違うのでここでは細かくは説明しませんが、おおまかなことだけ説明します。
1.ルーターの設定画面に行く(192.168.1.1)とかにアクセス
2.ルーターの設定画面で開放
おわり

外からアクセスしてみる

ターミナルで
curl ipinfo.io
を打つとグローバルアドレスが確認できる。
それをブラウザに手打ち。
ドメインを持っているのであれば、ドメインを指定してもよい。
